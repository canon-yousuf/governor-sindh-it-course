# Capstone: The Autonomous Humanoid

For the final project, you must demonstrate a fully autonomous behavior on a simulated humanoid.

## Requirements

1.  **Simulated Environment**: Must use Isaac Sim or Gazebo.
2.  **Task**: The robot must navigate from Point A to Point B, identify an object, and perform a manipulation task (e.g., push a button).
3.  **Code**: Must use ROS 2 and include a custom behavior tree.

## Grading Rubric

| Criteria | Points | Description |
| :--- | :--- | :--- |
| **Navigation** | 30 | Smooth path planning without collisions. |
| **Perception** | 30 | Correctly identifying the target object. |
| **Stability** | 20 | The humanoid must not fall over. |
| **Documentation** | 20 | Clear README and video demo. |
